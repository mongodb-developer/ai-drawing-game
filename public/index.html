    How It Works

    This game uses cutting-edge technologies to analyze your drawings:

    MongoDB Vector Search

    We use MongoDB Vector Search to compare your drawing to the prompt. It converts the labels of your drawing into a vector and finds the most similar vectors in our database.

    Learn more about MongoDB Vector Search

    AWS Rekognition

    AWS Rekognition analyzes your drawing to identify objects and scenes. It provides labels that we use to create the vector representation of your drawing.

    Learn more about AWS Rekognition
